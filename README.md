# vue-jdCart

v 1.0.0 介绍

这是用vue, vue-resouce, bootstrap实现的一个简单的京东购物车, 算是对vue的一个简单入门.

实现的功能：
  1. 选中或者不选中商品或者全选或者不全选商品
  2. 价格对应着选中商品的数量和价钱
  3. 删除商品
